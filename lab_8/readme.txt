Для запуска сервера необходимо ввести в терминал команду:

./srvr <root_directory>

root_directory - каталог, в котором будет запущен сервер

Для запуска клиента в другом терминале необходимо ввести команду:

./clnt

После старта на клиенте необходимо ввести IP-адресс сервера. Так как в рамках данной лабораторной
работы сервер и клиент запускаются на одной машине, по умолчанию вводится 127.0.0.1

По окончании работы клиента завершение проиходит при вводе опции QUIT

Для корректного завершения работы сервера необходимо нажать комбинацию клавиш Ctrl+C

Команды клиента для работы с сервером:

HELP — вывод справки
ECHO — эхо-запрос, возвращающий полученное от клиента без изменения
INFO — получение общей информации о сервере
CD   — смена текущего каталога
LIST — список файловых объектов текущего каталога
QUIT — завершение сеанса
