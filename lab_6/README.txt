# Instructions

# Run program

  Соберите проект
  ```
  make clean
  ```
  make
  ```
  
  Запустите программу генерирования файла, соблюдая нижеприведенные условия. Программа сгенерирует и выведет в консоль файл
  
- !!! Размер индекса в записях должен быть кратен 256 и кратно превышать планируемую
выделенную память для отображения (в рамках данной лабораторной минимальный размер выделенной памяти для отображения = 4096)
- !!! При большом количестве записей может быть задержка работы несколько секунд, так как необходимо время для генерирования файла большого размера
  ```
  ./generator <file size> <file name>
  ```
  
  Запустите программу сортировки файла, вводя необходимые данные (название файла указывается из предудыдущего пункта). Программа отсортирует
и выведет в консоль файл

- !!! В рамках данной лабораторной минимальный размер выделенной памяти для отображения = 4096
- !!! Количество блоков должно быть степенью двойки и превышать количество потоков
  ```
  ./sort_index <memory size> <blocks amount> <threads amount> <file name>
  ```
